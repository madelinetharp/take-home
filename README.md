## Setup Instructions

1. Clone the repository

   ```bash
    gh repo clone madelinetharp/take-home
   ```

2. Navigate to the project directory

   ```bash
    cd take-home
   ```
3. Install dependencies
   ```bash
    npm install
   ```
4. Launch Expo
   ```bash
    npx expo start
   ```
